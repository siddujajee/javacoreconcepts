Tree:
A hierarchical datastrcutre which consists of nodes and and subsequent child nodes connected to it.

Bindary Tree:
A tree is called binary tree if each node has at max of 2 child nodes.

Types Of Trees:

1. Full Binary Tree
2. Complete Binary Tree
3. Perfect Binary Tree
4. Balanced Binary Tree
5. Degenerative Binary Tree

6. Fully Binary Tree:
   binary tree's all the nodes will have either 0 or 2 sub nodes

7. Complete Binary Tree:
   when each level of tree is filled except the last level, and the last level nodes should have the left node only

8. Perfect Binary Tree:
   a binary tree where all the leaf nodes will be at the same level.

9. Perfect Binary Tree:
   a binary tree whose height is equals to log(n) where n is no. of nodes

10. Degenerative Binary Tree:
    a binary tree in which all the nodes will have exactly one child node.
